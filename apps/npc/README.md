## 使用说明

当前提交仅包含一个固定数字版本目录。安装前请先准备可连接的 NPS 服务端地址、客户端连接密钥，并确认服务端与客户端的 TLS 设置保持一致。

NPC 使用 `host` 网络模式，作为 NPS 客户端运行。它本身不提供固定 Web 端口，运行状态请在 1Panel 应用页面或容器日志中查看。

上游镜像中 `v0.27.01` 标签是较早构建，当前使用仍在更新的 `v0.26.x` 镜像维护线。

## 产品介绍

**NPC** 是 NPS 的客户端组件，用于连接 NPS 服务端，并根据服务端配置建立内网穿透代理。

## 主要功能

- **连接 NPS 服务端**：通过服务端地址和连接密钥接入；
- **TLS 通信**：可按服务端配置启用 TLS；
- **配合 NPS 管理**：客户端接入后可在 NPS Web 管理界面中配置和管理代理规则。
