## 产品介绍

DBX 是一个轻量级跨平台数据库客户端，可通过 Docker 自托管 Web 版本，支持 MySQL、PostgreSQL、SQLite、Redis、MongoDB、DuckDB、ClickHouse、SQL Server、Oracle、Elasticsearch 等 60+ 种数据库。

## 主要功能

- SQL 编辑、语法高亮、自动补全和格式化
- 数据浏览、编辑以及 CSV、Excel、JSON、Parquet 导入导出
- ER 图、结构对比、执行计划和字段血缘
- Redis 键浏览与 MongoDB 文档管理
- 内置 AI SQL 助手，支持 OpenAI 兼容接口
- 支持 SSH 隧道、深色模式和中英文界面
