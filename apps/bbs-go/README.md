## 产品介绍

**bbs-go** 是一个轻量级社区和问答平台，适合搭建论坛、知识库和讨论社区。

它将论坛讨论、问答流程、文章发布、评论互动、站内消息、内容治理、权限管理和后台运营整合在一起，适合需要长期沉淀内容、组织社区知识、维护用户互动的自托管场景。

一句话概括：**轻量搭建论坛、问答、知识库和讨论社区**。

## 为什么选择 bbs-go

- **开箱可用**：论坛、问答、文章、评论、点赞收藏、关注消息等核心社区能力可直接使用。
- **轻量完整**：适合搭建论坛、知识库、问答社区和讨论社区，不需要引入沉重的企业级系统。
- **增长闭环**：内置任务、积分、等级、勋章，支持用户活跃和长期留存。
- **运营友好**：提供内容治理、用户治理、权限治理与系统配置能力，方便持续运营。
- **双语支持**：内置 `zh-CN` / `en-US`，适合面向不同语言用户的社区场景。

## 主要功能

### 用户侧

- 账号注册与登录，支持多种登录方式
- 用户资料维护与个人主页展示
- 关注、粉丝关系管理
- 站内消息与互动提醒
- 积分记录与排行榜

### 内容侧

- 支持帖子、动态、文章发布与编辑
- 评论、回复、点赞、收藏等完整互动链路
- 标签与节点管理，便于内容组织和发现
- 支持投票、隐藏内容等互动玩法
- 站内搜索能力，提升内容检索效率

### 增长侧

- 每日签到，持续活跃激励
- 任务体系，支持新手任务、每日任务和成就任务
- 积分与经验奖励机制
- 等级成长配置
- 勋章与荣誉体系

### 运营侧

- 用户、帖子、评论、文章等统一治理
- 举报处理与违禁词管理
- 角色、菜单、接口权限分配
- 系统参数与站点配置管理
- 运营日志与行为留痕

## 适用场景

- 技术交流社区
- 问答社区
- 知识库
- 兴趣爱好社群
- 产品用户社区
- 企业内部知识社区
- 内容型会员社区

## 安装说明

- 安装前请先在 1Panel 中准备 MySQL 或 MariaDB 服务，并在安装表单中选择该数据库服务。
- 数据库需要为空库，安装向导会初始化表结构和基础数据。
- 应用启动后访问站点地址进入 `/install`，按页面提示完成站点和管理员初始化。
- 初始化完成后可访问 `/dashboard` 进入后台管理。
