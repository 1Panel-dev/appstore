## 产品介绍

**bbs-go** 是一个轻量级社区和问答平台，适合搭建论坛、知识库、问答社区和讨论社区。

## 主要功能

- 论坛、问答、文章、评论、点赞、收藏、关注和站内消息
- 用户、帖子、评论、文章、举报、违禁词和系统配置管理
- 角色、权限、运营日志和后台管理能力
- 任务、积分、等级和勋章等社区增长功能
- 内置 `zh-CN` 和 `en-US` 语言支持

## 安装说明

- 安装前请先在 1Panel 中准备 MySQL 或 MariaDB 服务。
- 数据库需要为空库，安装向导会初始化表结构和基础数据。
- 安装完成后访问站点地址进入 `/install` 完成初始化。
