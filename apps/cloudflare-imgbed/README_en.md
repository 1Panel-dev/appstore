## Introduction

**CloudFlare ImgBed** is an open-source file hosting solution supporting Docker and serverless deployment, with storage channels such as Telegram, Discord, Cloudflare R2, S3, Huggingface, and WebDAV.

## Features

- **File Lifecycle Management**: Provides upload, management, read, and delete capabilities.
- **Multiple Storage Channels**: Supports Telegram, Discord, Cloudflare R2, S3, Huggingface, WebDAV, and other storage channels.
- **APIs and Protocols**: Provides RESTful APIs and WebDAV support.
