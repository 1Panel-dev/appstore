## 产品介绍

**Audiobookshelf** 是一个自托管的有声读物和播客服务器。

## 主要功能

- 完全开源，包括Android 和 iOS 应用程序 （测试版）
- 即时传输所有音频格式
- 搜索并添加播客以下载带有自动下载的剧集
- 具有自定义权限的多用户支持
- 保持每个用户的进度并跨设备同步
- 自动检测库更新，无需重新扫描
- 使用批量上传拖放文件夹上传书籍和播客
- 备份您的元数据+每日自动备份
- 渐进式网络应用程序 (PWA)
- Web 应用程序和 Android 应用程序上的 Chromecast 支持
- 从多个来源获取元数据和封面艺术
- 章节编辑器和章节查找（使用Audnexus API）
- 将您的音频文件合并为一个 m4b
- 将元数据和封面图像嵌入到音频文件中（使用Tone）
- 基本电子书支持和电子阅读器
  - Epub、pdf、cbr、cbz
  - 将电子书发送到设备（即 Kindle）
- 开放播客和有声读物的 RSS 源
