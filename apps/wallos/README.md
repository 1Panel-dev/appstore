## 使用

第一次运行 wallos 时，必须创建一个帐户。
转到设置并个性化您的头像并添加您的家庭成员。同时添加/删除任何类别和货币。
从[Fixer](https://fixer.io/#pricing_plan)获取免费的 API 密钥并将其添加到设置中。 
如果您想触发汇率更新，请在添加 API 密钥后更改您的主要货币，然后将其改回您首选的货币。
