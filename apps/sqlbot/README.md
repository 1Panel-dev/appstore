## 产品介绍

**SQLBot** 是一款基于大模型和 RAG 的智能问数系统。

## 优势

- **开箱即用**：只需配置大模型和数据源即可开启问数之旅，通过大模型和 RAG 的结合来实现高质量的 text2sql；
- **易于集成**：支持快速嵌入到第三方业务系统，也支持被 n8n、MaxKB、Dify、Coze 等 AI 应用开发平台集成调用，让各类应用快速拥有智能问数能力；
- **安全可控**：提供基于工作空间的资源隔离机制，能够实现细粒度的数据权限控制。
