## Introduction

**Valkey** is a high-performance open-source data structure server designed primarily for key/value workloads. It supports a wide range of native data structures and an extensible plugin system.

## Features

- **Key-value and data structure workloads**: Suitable for caching, session storage, queues, and other high-frequency read/write scenarios.
- **High performance**: Built for high throughput and low-latency data access.
- **Extensible architecture**: Supports rich native data structures and an extensible plugin system for new access patterns.
- **Ecosystem friendly**: Provides familiar components such as `valkey-server` and `valkey-cli` for testing and integration.
