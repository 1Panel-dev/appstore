## 产品介绍

Music Tag Web 是面向远程服务器和 NAS 音乐库的 Web 音乐标签编辑器，适合在浏览器中直接管理歌曲元数据。官方项目支持 FLAC、APE、WAV、AIFF、MP3、M4A、OGG、OPUS、DSF、MP4 等多种音频格式。

## 主要功能

- 编辑标题、专辑、艺术家、歌词、封面等标签信息
- 支持批量刮削、批量改标签和音乐指纹识别
- 支持按艺术家、专辑或自定义规则整理音乐文件
- 支持繁简转换、文本替换、文件名解析与元数据补全
- 支持 ffmpeg 转码、整轨切割、封面导出与自定义上传
- 适合远程、NAS 和移动端场景访问
