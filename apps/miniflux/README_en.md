## Product Introduction

**Miniflux** is an "assertive" open-source feed reader. It doesn't pursue a fancy interface, but focuses on **minimalism, speed, privacy**. It is like a sharp Swiss Army knife, only doing reading and doing it to the extreme.

## Key Features

* **Wide Support**: Compatible with RSS, Atom, JSON feeds; Support OPML import and export.
* **Force Extraction**: Built-in Readability parser, one-click grabbing of full text without jumping to web pages.
* **All-round control**: Powerful keyboard shortcut support, no pressure to operate blindly.
* **Deep Integration**: Support 25+ third-party services (such as Wallabag, Instapaper, Pocket) and Fever/Google Reader API.
* **Powerful Search**: Full-text search based on PostgreSQL to find historical articles in seconds.
