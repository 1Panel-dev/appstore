## 产品介绍

**Miniflux** 是一款“有主见”的开源 Feed 阅读器。它不追求花哨的界面，而是专注于**极简、极速、隐私**。它像一把锋利的瑞士军刀，只做阅读这件事，且做到极致。

## 主要功能

* **广泛支持**：兼容 RSS、Atom、JSON Feed；支持 OPML 导入导出。
* **原力提取**：内置 Readability 解析器，一键抓取全文，无需跳转网页。
* **全能操控**：强大的键盘快捷键支持，盲操无压力。
* **深度集成**：支持 25+ 第三方服务（如 Wallabag, Instapaper, Pocket）及 Fever/Google Reader API。
* **强大搜索**：基于 PostgreSQL 的全文搜索，秒找历史文章。
