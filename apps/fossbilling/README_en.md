# FOSSBilling

FOSSBilling is an open-source billing and invoicing solution for service providers. It provides a complete billing platform for managing customers, subscriptions, invoices, and payments.

## Features

- **Customer Management**: Manage customer accounts and profiles
- **Billing & Invoicing**: Automated billing and invoice generation
- **Payment Processing**: Support for multiple payment gateways
- **Subscriptions**: Create and manage recurring subscriptions
- **Service Management**: Manage services and their pricing
- **Reports**: Comprehensive reporting and analytics
- **Multi-currency**: Support for multiple currencies
- **API**: RESTful API for integration

## Prerequisites

- Docker & Docker Compose
- MySQL or MariaDB database service

## Installation

1. Access the FOSSBilling web interface at `http://your-server:port`
2. Follow the installation wizard
3. Configure your billing settings and payment gateways
4. Start managing your business

## Configuration

Configure the following through the web interface:
- Admin credentials
- Company information
- Payment methods
- Email templates
- Tax settings
- Service offerings
