# FOSSBilling

FOSSBilling 是一个开源的计费和发票解决方案，专为服务提供商设计。它提供了一个完整的计费平台，用于管理客户、订阅、发票和支付。

## 功能特性

- **客户管理**: 管理客户账户和个人资料
- **计费与发票**: 自动化计费和发票生成
- **支付处理**: 支持多个支付网关
- **订阅管理**: 创建和管理循环订阅
- **服务管理**: 管理服务和其定价
- **报表分析**: 全面的报表和分析功能
- **多货币支持**: 支持多种货币
- **API接口**: RESTful API 集成

## 前置要求

- Docker & Docker Compose
- MySQL 或 MariaDB 数据库服务

## 安装

1. 访问 FOSSBilling 网络界面 `http://你的服务器:端口`
2. 按照安装向导进行操作
3. 配置你的计费设置和支付网关
4. 开始管理你的业务

## 配置

通过网络界面配置以下项目：
- 管理员凭证
- 公司信息
- 支付方式
- 邮件模板
- 税务设置
- 服务定价
