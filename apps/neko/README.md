## 产品介绍

Neko 是一个基于 WebRTC 的开源虚拟浏览器 / 远程桌面流媒体服务。
它允许你在服务器上运行浏览器或桌面环境，并通过浏览器实时访问，就像把一个远程电脑共享给所有人一样。

你可以使用 Neko 远程浏览网页、观看视频、协同操作桌面应用，甚至将它当作隐私浏览器或多人共享浏览的工具。它就像你的“云端浏览器”！

## 特性

- 支持运行 完整浏览器环境（Firefox、Chromium、Chrome、Edge、Brave 等）
- 支持运行 完整 Linux 桌面环境（XFCE、KDE 等）
- 采用 WebRTC 传输，低延迟、高质量的图像与音频流
- 支持 多人同时在线协作，可设置权限、控制权、房间管理
- 可作为 共享浏览器、远程教学、远程协作、共同观影等多种用途
- 支持 音频共享，多人可一起观看视频、听音乐
- 镜像支持 即时转码（编码），可根据带宽自动调整质量
- 资源占用低，在轻量服务器上也可流畅运行
- 完全 自托管，隐私数据只存储在你的服务器中
- 提供官方 Docker 镜像，一条命令快速部署
- 支持多架构，包括 x86_64 / ARM / ARM64（如 Raspberry Pi）
- 可自定义安装应用（媒体播放器、开发工具等）
- 自动 监控服务状态，自动重连，提升使用稳定性
- 支持多种语言 的现代化 Web 界面
