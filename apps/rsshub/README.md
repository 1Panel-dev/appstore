## 产品介绍

RSSHub 是一款开源、可自行部署的通用 RSS 内容聚合器。它将万物皆可 RSS 的理念变为现实，能够从各类网站、社交媒体、新闻平台等数千种数据源中抓取内容并生成标准 RSS 输出。RSSHub 拥有全球超过 5,000 个实例组成的庞大网络，每天为海量用户输送聚合内容。其充满活力的开源社区持续贡献新的路由、功能和修复，确保覆盖范围不断扩展。RSSHub 与 AI RSS 阅读器 Folo 搭配使用效果尤佳，可实现 Feed 发现与现代阅读工作流的无缝融合。

## 主要功能

- 万物皆可 RSS：支持从数千个网站和平台生成 RSS 订阅源，覆盖社交媒体（Twitter、Instagram、YouTube、抖音/TikTok 等）、新闻、博客、论坛等
- 开箱即用的路由库：拥有海量预定义路由规则，持续由社区维护和更新
- 灵活部署：支持 Docker、Vercel、Cloudflare Workers、Nix、Fly.io 等多种部署方式，便捷高效
- 浏览器扩展支持：配套 RSSHub Radar 扩展，帮助用户快速发现当前网站支持的 RSS 和 RSSHub 路由
- 跨平台生态：提供 iOS 端 RSSBud 和 Android 端 RSSAid 等移动端辅助工具
- 支持自定义路由：用户可根据需求编写和扩展自己的 RSS 路由规则
- 强大的社区生态：拥有丰富的文档、Telegram 群组和频道、以及精选路由集合（Awesome RSSHub Routes）
- 可作为 PWA 安装使用，支持现代化阅读体验
- 与 AI RSS 阅读器 Folo 深度集成，支持智能 Feed 发现与阅读
- 基于 AGPL-3.0 开源协议，代码完全公开，支持自托管保障数据隐私
