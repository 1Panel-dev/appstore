# 使用说明

## 1. 数据库连接问题

- 1、进入数据库菜单
- 2、新建 discuz 数据库（初始化时需要）
- 3、点击连接信息按钮获取数据库连接地址，默认为：
> mysql:3306

## 2. 持久化问题

应用以存储卷方式进行存储数据
