## 产品介绍

**EasyTier** 是一个简单、安全、去中心化的内网穿透 VPN 组网方案，使用 Rust 与 Tokio 实现，适合用于异地设备互联、家庭实验室、办公网络打通和子网代理等场景。

## 核心特性

- **去中心化组网**：无需依赖中心化控制节点，节点之间可以平等互联。
- **安全通信**：支持 WireGuard 加密通信，也支持使用 AES-GCM 保护中转流量。
- **高性能转发**：采用全链路零拷贝设计，兼顾低延迟与高吞吐。
- **复杂网络适应性**：支持 UDP NAT 穿透、TCP 回退、多路径与自动切换，适应复杂网络环境。
- **多协议支持**：支持 TCP、UDP、WebSocket、QUIC 等多种节点间通信协议。
- **跨平台部署**：支持 Linux、macOS、Windows 和 Android，便于多设备混合组网。
- **子网代理能力**：支持将局域网网段暴露给 VPN 子网，满足点对网访问需求。

## 使用说明

- EasyTier 在 1Panel 中以 `host` 网络模式运行，并默认启用特权模式，以支持 TUN 设备与底层网络能力。
- 安装完成后，请优先编辑安装目录中的 `data/config.toml`，按实际需求配置网络名称、网络密钥、监听地址、对等节点和子网代理等参数。
- 同目录下提供了 `data/config-example.toml`，可作为创建和调整节点配置时的参考模板。
- 如果需要让多个节点加入同一个虚拟网络，请确保各节点的 `network_name` 与 `network_secret` 配置一致，并按需填写 `peer` 节点信息。
- 修改配置文件后，重启应用即可使配置生效；运行中的详细状态也可以通过容器日志查看。
