## 产品介绍

FileCodeBox 是一个轻量、现代的自托管文件分享工具。无需注册，上传文件或文本后即可获得取件码，对方输入取件码即可获取内容。

## 主要特色

- [x] 轻量简洁：FastAPI + SQLite + Vue 3
- [x] 轻松上传：支持拖拽、粘贴、批量和分片上传
- [x] 多种类型：统一分享文本和文件
- [x] 防止爆破：错误次数限制
- [x] 防止滥用：IP 上传频率限制
- [x] 口令分享：随机口令，存取文件，自定义次数以及有效期
- [x] 国际化：支持中文和英文
- [x] 匿名分享：无需注册，无需登录
- [x] 管理面板：文件、分享、存储和安全设置集中管理
- [x] 存储配额：可设置全局存储容量上限
- [x] 多存储后端：内置本地、S3 和 WebDAV，可通过可选依赖扩展 OneDrive 与 OpenDAL
- [x] 一键部署：基于官方多架构 Docker 镜像部署
