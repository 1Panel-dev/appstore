## 产品介绍

**Kibana** 是 Elastic Stack 的数据分析与可视化平台，用于浏览、搜索和分析 Elasticsearch 中的数据。它通过浏览器提供统一界面，适合日志分析、可观测性、安全事件分析和业务数据展示等场景。

## 主要功能

- **数据探索**：支持对 Elasticsearch 数据进行搜索、过滤和交互式分析。
- **可视化仪表板**：可将图表、指标、地图和表格组合成仪表板，便于统一展示和共享。
- **日志与可观测性分析**：适用于日志查询、指标观察和故障排查等常见运维分析场景。
- **安全分析**：可用于查看安全事件、调查异常行为并辅助风险分析。
- **Elastic Stack 管理**：帮助用户查看和管理 Elastic Stack 中的数据与相关能力。
