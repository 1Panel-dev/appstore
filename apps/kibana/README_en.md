## Introduction

**Kibana** is the data analytics and visualization platform for the Elastic Stack. It helps you browse, search, and analyze data stored in Elasticsearch through a browser-based interface, making it suitable for log analysis, observability, security investigations, and business data dashboards.

## Features

- **Data exploration**: Search, filter, and interactively analyze data stored in Elasticsearch.
- **Visualization dashboards**: Combine charts, metrics, maps, and tables into dashboards for unified display and sharing.
- **Logs and observability analysis**: Useful for log investigation, metric exploration, and troubleshooting workflows.
- **Security analysis**: Review security events, investigate unusual behavior, and support risk analysis.
- **Elastic Stack management**: Helps users work with data and related capabilities across the Elastic Stack.
