## 产品介绍

**Harness Open Source** 是一个开源开发平台，集成了代码托管、自动化 DevOps 流水线、托管开发环境（Gitspaces）以及制品仓库等功能。

## 主要特性

- **统一代码仓库管理**：支持创建 Git 仓库、代码评审、分支保护，以及 commit 秘密扫描等安全措施。  
- **预配置开发环境（Gitspaces）**：在云端快速启动可共享、带依赖环境的开发环境，无需复杂本地搭建。  
- **自动化 CI/CD 流水线**：内置多个可复用模板，实现代码推送/PR/分支触发构建、测试、部署等流程自动化。  
- **集中化制品仓库**：集中管理 Docker 镜像、Helm Charts 等制品，支持 artifact 推送、拉取与审计。  
- **安全与合规控制**：集成 Secret 扫描、漏洞扫描、供应链安全等机制，在开发生命周期各阶段建立安全防护。  
- **丰富集成与迁移支持**：支持与 GitHub、GitLab、Bitbucket 等工具的集成，并提供一键迁移功能利于现有项目切换。
