## 使用说明

当前提交仅包含一个固定数字版本目录。安装时请按表单设置 Web 管理端口、服务端客户端通信端口、TLS 通信端口、HTTP/HTTPS 域名代理端口、Web 管理账号密码和客户端连接密钥。

NPS 使用 `host` 网络模式，相关端口会直接监听在宿主机上。安装前请确认端口未被占用，并根据需要放行防火墙或安全组。

默认包不内置 TLS 证书私钥。如需启用 Web HTTPS 或 HTTPS 代理证书，请在安装后的 `conf` 目录中放置对应证书文件，并按 NPS 配置要求调整证书路径。

上游镜像中 `v0.27.01` 标签是较早构建，当前使用仍在更新的 `v0.26.x` 镜像维护线。

## 产品介绍

**NPS** 是一款轻量级、高性能、功能强大的内网穿透代理服务器，支持 TCP、UDP、HTTP、HTTPS、SOCKS5 等多种代理方式，适合将内网服务安全地暴露到外部网络。

## 主要功能

- **多协议代理**：支持 TCP、UDP、HTTP、HTTPS、SOCKS5 等代理模式；
- **Web 管理界面**：可通过 Web UI 管理客户端、隧道和域名代理；
- **客户端密钥连接**：NPC 客户端可使用服务端配置的连接密钥接入；
- **TLS 通信**：支持服务端与客户端之间启用 TLS 通信。
