# ZPan

ZPan 是一个开源的 S3 原生文件托管平台，适用于网盘、文件分享、图床和直接上传到对象存储的工作流。

ZPan 服务端负责认证、元数据、分享、配额、团队空间、WebDAV 和后台管理。文件上传通过预签名 URL 由浏览器直接传到 S3 兼容存储，避免占用服务器上传带宽。

安装后请先访问应用地址注册并登录，然后进入 **管理后台 -> 存储** 添加 S3 兼容存储。存储端点必须能被浏览器访问，因为文件会从客户端直接上传到对象存储。

支持的对象存储包括 Cloudflare R2、AWS S3、Backblaze B2、MinIO、RustFS、Tigris 以及其他 S3 兼容服务。
