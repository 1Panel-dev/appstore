# PerfreeBlog

**PerfreeBlog** 是一款基于java开发的博客/CMS建站平台,拥有丰富的主题支持及扩展插件功能,给您带来全新的创作体验~

## 功能简介

- 设计简洁，界面美观
- 支持多主题自由切换
- 强大的主题在线编辑功能, 支持主题在线编辑
- 支持自定义页面
- 支持友情链接
- 支持附件管理,附件在线预览
- 支持扩展插件,插件编写方式与普通程序基本一致,直接后台安装生效,丰富程序多样性
- 主题开发简单快速,多种自定义指令及api接口支持
- 支持邮件服务,评论信息邮件提醒
- 安装部署简单
- ......
