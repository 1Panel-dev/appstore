# XiaoMusic

## 产品介绍

XiaoMusic 是一个使用小爱音箱播放音乐的开源项目，音乐使用 yt-dlp 下载，实现无限听歌，解放小爱音箱的功能。

## 主要功能

- **无限听歌**：通过 yt-dlp 下载音乐，实现无限听歌功能
- **小爱音箱控制**：支持语音口令控制音乐播放
- **Web 管理界面**：提供友好的 Web 界面进行配置和管理
- **多平台支持**：支持 Docker 部署，兼容各大 NAS 平台
- **自定义口令**：可以完全自定义语音口令，支持插件扩展

### 语音口令功能

- **播放歌曲**：播放本地的歌曲
- **播放歌曲+歌名**：比如：播放歌曲周杰伦晴天
- **上一首**：切换到上一首歌曲
- **下一首**：切换到下一首歌曲
- **单曲循环**：循环播放当前歌曲
- **全部循环**：循环播放整个歌单
- **随机播放**：随机播放歌曲
- **关机/停止播放**：停止音乐播放
- **刷新列表**：当复制了歌曲进 music 目录后，刷新歌单
- **播放列表+列表名**：比如：播放列表其他
- **加入收藏**：把当前播放的歌曲加入收藏歌单
- **取消收藏**：把当前播放的歌曲从收藏歌单里移除
- **播放列表收藏**：播放收藏歌单
- **播放本地歌曲+歌名**：本地找不到也不会去下载
- **播放列表第几个+列表名**：指定播放列表中的特定歌曲
- **搜索播放+关键词**：搜索关键词作为临时搜索列表播放
- **本地搜索播放+关键词**：本地找不到也不会去下载

### 注意事项

- 初次配置时需要在页面上输入小米账号和密码保存后才能获取到设备列表
- 音乐文件会自动下载到 music 目录
- 配置文件保存在 conf 目录
- 遇到问题可以去 web 设置页面底部点击【下载日志文件】按钮，然后搜索日志文件内容确保里面没有账号密码信息后，在提 issues 反馈问题时把下载的日志文件带上
