## 产品介绍

**UPage** 是一款基于大语言模型的可视化网页构建平台，支持接入主流大模型，只需通过自然语言描述需求，即可快速生成个性化、高颜值的网页，让创作更高效、更智能。

## 主要功能

- **可视化编辑，所见即所得**：简洁直观的可视化编辑器，支持实时预览，轻松调整布局与样式；
- **多页面一键生成**：支持同时生成多个关联页面，快速搭建完整网站结构；
- **标准代码自由导出**：自动生成规范的 HTML/CSS/JS 代码，便于集成至现有项目或二次开发；
- **响应式设计，全端适配**：自动适配桌面、平板、移动端等多种设备，确保跨平台完美呈现。

## V2 版本亮点

- **Agent 驱动**：页面生成逻辑全面重构为 Agent 架构，搭配 React Router v7，交互更智能、更流畅；
- **视觉模型支持**：新增独立的视觉模型配置，当主模型不支持多模态时，可自动调用视觉模型处理图片；
- **AI 工具扩展**：新增 Serper 搜索、天气查询等多种 AI 工具，增强页面生成的信息获取能力；
- **主流设计系统参考**：生成页面前可引用主流设计系统，让输出更专业、更统一；
- **自动页面标题**：根据内容智能生成页面标题，告别「Untitled Page」；
- **网页补丁操作**：支持对已生成的网页进行局部修改和迭代优化。
