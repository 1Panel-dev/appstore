## Introduction

**Vector** is a high-performance, end-to-end observability data pipeline that can be deployed as either an Agent or an Aggregator. It helps you collect, transform, and route logs and metrics while keeping control of your observability data.

## Features

- **Reliable**: Built in Rust, with reliability as Vector's primary design goal.
- **End-to-end**: Can be deployed as both an Agent and an Aggregator.
- **Unified pipeline**: Handles logs and metrics in one pipeline, with tracing support continuing to evolve.
- **Flexible routing**: Route logs and metrics to the vendors you use today and keep flexibility for future changes.
- **Common use cases**: Reduce observability costs, improve data quality, gain better insights, and consolidate multiple agents.
