## 产品介绍

**Vector** 是一款高性能、端到端的可观测性数据管道工具，可作为 Agent 或 Aggregator 部署，帮助你收集、转换并路由日志和指标数据，从而更好地掌控自己的可观测性数据。

## 主要功能

- **可靠**：基于 Rust 构建，官方将可靠性作为 Vector 的首要设计目标。
- **端到端**：既可以作为 Agent 部署，也可以作为 Aggregator 部署。
- **统一数据管道**：统一处理日志与指标数据，跟踪能力仍在持续演进中。
- **灵活路由**：将日志和指标路由到你当前使用的厂商，也便于未来切换到其他厂商。
- **典型用途**：可用于降低可观测性成本、提升数据质量、增强洞察，并减少多种代理并存带来的复杂度。
