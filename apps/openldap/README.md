## 产品介绍

[OpenLDAP](https://openldap.org/) 是轻型目录访问协议（Lightweight Directory Access Protocol，LDAP）的自由和开源的实现。

该应用使用了 Bitnami 构建的 OpenLDAP Docker 镜像，通过该应用可以快速部署 OpenLDAP 服务。
