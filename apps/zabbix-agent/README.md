## 产品介绍

**Zabbix** 是一款企业级开源分布式监控解决方案，旨在监控网络设备、服务器、服务和其他 IT 资源的性能和可用性。

## 特性

- **资源发现**：自动发现网络实体、服务器资源，以及设备的上下线情况。可使用开箱即用的集成模板，对从低层设备到 SaaS 服务的任何资源进行监控。
- **指标采集**：通过 Agent 或无 Agent 的方式，从任意来源采集监控指标，包括设备、传感器、操作系统、虚拟化平台、容器平台（如 Docker、Kubernetes）、云基础设施、数据库、网页、Java 生态、应用服务器、API 端点、业务应用等几乎所有类型的系统。
- **根因分析与问题检测**：提供高性能的实时问题检测能力，能够对现有与新产生的问题进行关联分析，并执行根因定位。
- **事件、告警与通知**：当监控环境中出现新问题（无论是主动发现还是事后捕捉）时，可及时触发告警。支持多种消息通知方式，包括 Slack、JIRA、Microsoft Teams、邮件、短信等，用于告知系统内发生的不同类型事件。
- **“单一视图”监控体验**：将采集到的数据和事件通过图表、列表、地理地图、网络拓扑图等方式进行可视化展示。
- **多租户与分布式监控**：在多个数据中心、部门或组织间使用同一套监控解决方案，并支持在跨防火墙环境下对远端位置进行监控与远程命令执行。
- **极致灵活性**：可根据需求定制 Zabbix，并利用丰富的内置功能，如通过 HTTP 推送指标与事件、报表、审计、安全管理、服务 SLA 计算等。
