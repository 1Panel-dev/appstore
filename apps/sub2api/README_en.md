## Introduction

Sub2API is an open-source AI API gateway platform for distributing and managing subscription quotas from AI products. Platform-generated API keys provide access to upstream AI services while Sub2API centralizes authentication, billing, load balancing, and request forwarding.

## Features

- Supports multiple upstream account types, including OAuth and API keys
- Provides API key distribution, user management, and quota management
- Supports token-level usage tracking, billing, and rate limiting
- Supports account scheduling, sticky sessions, and request forwarding
- Includes a web administration dashboard and runtime monitoring
