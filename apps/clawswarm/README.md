## 产品介绍

**ClawSwarm** 是一个面向 OpenClaw 的开源协作编排系统，把群体智能带到多 Agent 协作场景中。它突破了传统 AI 一问一答的限制，让多个具备不同专长的 Agent 能够进入同一个群聊协作，适合用于集体讨论、任务拆解和自动执行。

## 核心特性

- **多 Agent 群聊协作**：开发、设计、测试、研究等不同角色的 Agent 可以在同一个会话中共同完成任务。
- **统一 Web 控制台**：通过浏览器管理 OpenClaw 实例、会话、群组、消息和调度任务。
- **OpenClaw 插件集成**：支持与 OpenClaw 进行联动，构建多智能体协作工作流。
- **容器化部署**：官方提供 Docker 镜像，安装后即可快速启动和持久化数据。

## 默认访问信息

- 访问地址：`http://<服务器IP>:端口`
- 默认用户名：`admin`
- 默认密码：`admin123456`

## 安装后建议

- 启动完成后，先通过默认管理员账号登录 ClawSwarm Web 界面。
- 如需与 OpenClaw 集成，可继续参考官方插件安装文档：
