## 产品介绍

**Hermes Agent** 是 Nous Research 构建的自我改进型 AI Agent。它内置学习循环，可以从使用经验中创建和改进技能，保留长期记忆，检索历史会话，并在多次会话之间持续建立对用户的理解。

Hermes Agent 不绑定在本地电脑上，既可以运行在 VPS、GPU 集群等环境中，也可以通过命令行、Web UI 或消息平台与用户交互。1Panel 安装包提供浏览器 Web UI，用于管理配置、API Key，并查看运行状态和会话信息。

## 核心特性

- **多模型支持**：支持 OpenAI、Kimi/Moonshot、MiniMax、z.ai/GLM、Xiaomi MiMo、OpenRouter 以及自定义端点，并可通过配置切换模型。
- **终端与 Web UI**：提供终端交互界面，同时包含基于浏览器的 Web UI，可用于配置管理、API Key 管理和会话状态查看。
- **消息网关**：支持通过微信、QQ、企业微信、钉钉、飞书、Telegram、Discord 等消息平台与 Agent 对话。
- **学习与记忆**：支持由 Agent 管理的记忆、历史会话搜索、技能创建与技能改进，用于跨会话保留上下文。
- **计划任务**：内置 Cron 调度能力，可将日报、备份、巡检等任务投递到已连接的平台。
- **工具与扩展**：提供工具集系统、MCP 集成和多种终端后端，适合需要工具调用、自动化和远程执行的场景。
- **OpenClaw 迁移**：支持从 OpenClaw 导入设置、记忆、技能和部分密钥配置。
