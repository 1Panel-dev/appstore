## 产品介绍

**Vane** 是一个 **注重隐私的 AI 问答引擎**，可以完全运行在你自己的硬件上。它结合来自互联网的海量知识，并支持 **本地 LLM（Ollama）** 以及云端 AI 提供商（如 OpenAI、Claude、Groq 等），在提供 **带来源引用的准确答案** 的同时，确保你的搜索过程完全私密。

## 功能特性

- **支持主流 AI 提供商** - 可通过 Ollama 使用本地 LLM，也可以连接 OpenAI、Anthropic Claude、Google Gemini、Groq 等云端模型，根据需要自由组合使用不同模型。
- **智能搜索模式** - 提供三种模式：Speed Mode（快速模式）用于快速获取答案，Balanced Mode（平衡模式）适合日常搜索，Quality Mode（高质量模式）用于深入研究。
- **自定义信息来源** - 可选择搜索网页、讨论内容或学术论文。更多数据源与集成正在开发中。
- **Widgets 小组件** - 当相关时自动显示实用的 UI 卡片，例如天气、计算结果、股票价格等快速信息。
- **基于 SearxNG 的网页搜索** - 聚合多个搜索引擎，同时保护你的隐私。未来还将支持 Tavily 和 Exa，以获得更好的搜索结果。
- **图片与视频搜索** - 在文本结果之外，还可以搜索图片和视频内容，搜索不再局限于文章。
- **文件上传** - 上传文档并进行问答。Vane 可以理解 PDF、文本文件和图片等多种文件类型。
- **指定站点搜索** - 当你知道目标网站时，可以将搜索限制在特定域名中，非常适合技术文档或学术研究。
- **智能搜索建议** - 在输入时提供智能搜索建议，帮助你更好地构建查询问题。
- **发现（Discover）** - 浏览当天有趣的文章和热门内容，即使不主动搜索也能获取新信息。
- **搜索历史** - 所有搜索记录都会保存在本地，方便随时回顾你的发现和研究内容。
- **更多功能即将推出** - 我们正在根据社区反馈持续开发新功能，欢迎加入我们的 Discord，一起参与 Vane 的未来发展！
