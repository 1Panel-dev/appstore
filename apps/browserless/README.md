## 产品介绍

**Browserless** 是一款强大的开源 headless 浏览器管理方案。它不仅是将 Chrome 搬进 Docker，更是一套成熟的浏览器即服务（Browser-as-a-Service）架构。它解决了在云端、CI/CD 或容器中运行浏览器的种种“噩梦”——如字体缺失、内存泄露、依赖地狱和僵尸进程，让你专注于自动化逻辑而非底层基础设施。

## 主要功能

* **全能兼容**：完美支持 **Puppeteer**、**Playwright** 和标准 WebDriver，无需修改代码，只需更改连接地址即可无缝迁移。
* **内置渲染**：提供 REST API，无需编写复杂的脚本即可实现一键获取 HTML、生成 PDF 或网页截图。
* **多平台支持**：原生支持 **ARM64** 架构（包括 Apple Silicon），在各类轻量级云服务器上表现优异。
* **开箱即用**：内置完整的系统字体与表情包支持，彻底告别渲染中文时的“豆腐块”乱码问题。
* **性能管理**：内置队列和并发控制机制，支持配置连接超时和健康检查，即便 Chrome 崩溃也不会影响服务稳定性。
* **调试利器**：内置 **Debug Viewer**，可以实时查看并交互正在运行的无头浏览器会话，让调试不再盲目。
* **AI 友好**：全新的 **MCP Server** 支持，可将 AI 助手（如 Claude, Cursor）直接连接到自动化流程中。
